## [0.1.0] - 2026-01-22

### Added
- Initial release of the Octoparse MCP Server
- Account management — view account level, balance, and subscription status
- Template search — find scraping templates using AI
- Task creation — create tasks from templates via natural language
- Task control — start, stop, and monitor cloud task execution
- Data export — download scraped data as JSON or CSV
- OAuth authentication support
